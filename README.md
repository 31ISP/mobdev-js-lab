## Запуск JS через Node.js

Node.js позволяет запускать JavaScript прямо в терминале, без браузера.

**Запуск файла:**
```bash
node имя_файла.js
```

```javascript
console.log(typeof "hello")      // string
```

## Раздел 1. Числа

### Теория

В JavaScript все числа — один тип `number`, неважно целое это или дробное.

**Базовая арифметика** работает стандартно: `+`, `-`, `*`, `/`. Оператор `%` возвращает остаток от деления, `**` — возведение в степень.

```javascript
console.log(10 % 3)   // 1 — остаток от деления
console.log(2 ** 8)   // 256 — два в восьмой степени
```

**`NaN` (Not a Number)** — специальное значение, которое возникает при некорректных математических операциях. Важная особенность: `NaN !== NaN`, поэтому для проверки используют `isNaN()`.

```javascript
console.log("текст" * 2)       // NaN
console.log(isNaN("текст" * 2)) // true
```

**`Math`** — встроенный объект с математическими функциями:

| Метод | Что делает |
|---|---|
| `Math.round(x)` | Округляет до ближайшего целого |
| `Math.floor(x)` | Округляет вниз |
| `Math.ceil(x)` | Округляет вверх |
| `Math.abs(x)` | Модуль числа |
| `Math.max(a, b, ...)` | Максимум из чисел |
| `Math.min(a, b, ...)` | Минимум из чисел |
| `Math.random()` | Случайное число от 0 до 1 |
| `Math.sqrt(x)` | Квадратный корень |

`toFixed(n)` — метод числа, возвращает строку с `n` знаками после запятой:

```javascript
const price = 19.9876
console.log(price.toFixed(2))  // "19.99"
```

**Преобразование строки в число:**

```javascript
Number("42")      // 42
parseInt("42px")  // 42 — берёт целую часть до нецифрового символа
parseFloat("3.14abc") // 3.14
```

## Раздел 2. Строки

### Теория

Строки в JS можно объявлять через `'одинарные'`, `"двойные"` или `` `обратные кавычки` ``. Обратные кавычки — это **шаблонные литералы**: в них можно вставлять выражения через `${}`.

```javascript
const name = "Мария"
const age = 21
console.log(`Студентка ${name}, возраст: ${age} лет`)
// Студентка Мария, возраст: 21 лет
```

Внутри `${}` можно писать любое JS-выражение, не только переменную:

```javascript
console.log(`Сумма: ${10 + 20}`)   // Сумма: 30
console.log(`Это ${age >= 18 ? "совершеннолетний" : "несовершеннолетний"}`)
```

**Полезные методы строк:**

| Метод | Что делает |
|---|---|
| `.length` | Длина строки (свойство, не метод) |
| `.toUpperCase()` | В верхний регистр |
| `.toLowerCase()` | В нижний регистр |
| `.trim()` | Убирает пробелы по краям |
| `.includes(str)` | Содержит ли подстроку → `true/false` |
| `.startsWith(str)` | Начинается ли с подстроки |
| `.endsWith(str)` | Заканчивается ли подстрокой |
| `.indexOf(str)` | Индекс первого вхождения, или `-1` |
| `.slice(start, end)` | Вырезает подстроку по индексам |
| `.replace(old, new)` | Заменяет первое вхождение |
| `.replaceAll(old, new)` | Заменяет все вхождения |
| `.split(sep)` | Разбивает строку в массив |
| `.repeat(n)` | Повторяет строку n раз |
| `.padStart(n, char)` | Дополняет строку слева до длины n |

```javascript
const str = "  Hello, World!  "
console.log(str.trim())                   // "Hello, World!"
console.log(str.trim().toLowerCase())     // "hello, world!"
console.log("ha".repeat(3))              // "hahaha"
console.log("7".padStart(3, "0"))        // "007"
```

Строки индексируются с нуля. Можно обращаться к символу через `str[i]` или `str.at(i)` — метод `at` поддерживает отрицательные индексы (`str.at(-1)` — последний символ).

## Раздел 3. Массивы

### Теория

Массив — упорядоченный список значений. Индексация с нуля.

```javascript
const fruits = ["яблоко", "банан", "вишня"]
console.log(fruits[0])      // "яблоко"
console.log(fruits.length)  // 3
```

**Методы изменения массива:**

| Метод | Что делает |
|---|---|
| `.push(el)` | Добавляет в конец |
| `.pop()` | Удаляет и возвращает последний |
| `.unshift(el)` | Добавляет в начало |
| `.shift()` | Удаляет и возвращает первый |
| `.splice(i, n)` | Удаляет `n` элементов начиная с `i` |

**Методы перебора — возвращают новый массив:**

`.map(fn)` — преобразует каждый элемент. Возвращает новый массив той же длины.

```javascript
// Каждое число возводим в квадрат
const nums = [1, 2, 3, 4]
const squared = nums.map(n => n ** 2)
console.log(squared)  // [1, 4, 9, 16]
```

`.filter(fn)` — оставляет только те элементы, для которых функция вернула `true`.

```javascript
// Только чётные числа
const evens = nums.filter(n => n % 2 === 0)
console.log(evens)  // [2, 4]
```

`.reduce(fn, initial)` — сворачивает массив в одно значение. Функция принимает аккумулятор и текущий элемент.

```javascript
// Сумма всех элементов
const sum = nums.reduce((acc, n) => acc + n, 0)
console.log(sum)  // 10
```

**Другие полезные методы:**

| Метод | Что делает |
|---|---|
| `.find(fn)` | Возвращает первый подходящий элемент |
| `.findIndex(fn)` | Индекс первого подходящего |
| `.some(fn)` | Хоть один подходит? → `true/false` |
| `.every(fn)` | Все подходят? → `true/false` |
| `.includes(el)` | Содержит элемент? → `true/false` |
| `.sort(fn)` | Сортирует массив (изменяет исходный!) |
| `.reverse()` | Разворачивает массив |
| `.join(sep)` | Объединяет в строку |
| `.flat()` | Разворачивает вложенные массивы |
| `.slice(start, end)` | Копирует часть массива |

> ⚠️ **Важно:** `sort()` без аргументов сортирует как строки. Для чисел передавайте функцию:
> ```javascript
> [10, 2, 1, 20].sort((a, b) => a - b)  // [1, 2, 10, 20]
> ```

## Раздел 4. Объекты

### Теория

Объект — набор пар «ключ: значение». Значением может быть что угодно, включая функции (тогда это называется **методом** объекта).

```javascript
const student = {
  name: "Алексей",
  age: 20,
  grade: 4.5,
  greet() {
    return `Привет, я ${this.name}`  // this — это сам объект
  }
}

console.log(student.name)      // "Алексей" — через точку
console.log(student["age"])    // 20 — через скобки (полезно для динамических ключей)
console.log(student.greet())   // "Привет, я Алексей"
```

**Работа с ключами и значениями:**

```javascript
Object.keys(student)    // ["name", "age", "grade", "greet"]
Object.values(student)  // ["Алексей", 20, 4.5, ƒ]
Object.entries(student) // [["name", "Алексей"], ["age", 20], ...]
```

**Деструктуризация** — удобный способ извлечь значения из объекта в переменные:

```javascript
const { name, age } = student
console.log(name)  // "Алексей"
console.log(age)   // 20

// Можно сразу переименовать:
const { name: studentName } = student
console.log(studentName)  // "Алексей"

// Можно задать значение по умолчанию:
const { city = "Москва" } = student
console.log(city)  // "Москва" — ключа нет, используется дефолт
```

**Spread-оператор `...`** — копирует свойства одного объекта в другой:

```javascript
const base = { role: "user", active: true }
const admin = { ...base, role: "admin", name: "Петр" }
// { role: "admin", active: true, name: "Петр" }
```

**Добавление и удаление свойств:**

```javascript
student.email = "alex@mail.ru"  // добавить
delete student.grade            // удалить
```

**Проверка наличия ключа:**

```javascript
console.log("name" in student)              // true
console.log(student.hasOwnProperty("age"))  // true
```
